options

Return Value

Discussion

Consider this example:

VKRequest *request = [[VKRequest alloc] initWithMethod:@“users.get” options:@{@“fields”: @“nickname,bdate,status”}];

Object will be created VKRequest and subsequently implemented the method call users.get the social network. Option fields will be equal to “nickname,bdate,status” and therefore social network will return the nickname, date of birth and the status of the current user.
