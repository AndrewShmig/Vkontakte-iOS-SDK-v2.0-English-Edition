VKCachedData Class Reference

Inherits from NSObject
Declared in VKCachedData.h

Overview

This interface is intended for storing, retrieving and removing cache requests. Data will be stored on local drive and in the directory set during initialization process

cachedDataForURL:offlineMode:

Retrieve cached data which matches to passed url. If associated item does not exist nil will be returned

- (NSData *)cachedDataForURL:(NSURL *)url offlineMode:(BOOL)offlineMode

Parameters

url

url which matches to cached data

offlineMode

cache access offline mode

Return Value

NSData instance

Discussion

If offlineMode is passed as “YES” the data will be returned from the cache even if it’s expired. If offlineMode is equal to “NO” the cached item will be deleted if it’s expired

The offlineMode parameter is useful when there is no internet connection
